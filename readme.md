## Build

### Linux
* Run ./deps.sh in command line.
* Install Qt.
* Run the following command.
  ```
  make
  ```

### Windows
* Install Qt(MinGW).
* Run the following command.
  ```
  build-windows.bat
  ```

### Android
* Run the following commands.
  ```
  source ../android-build/android.profile
  ./android-build.sh
  ```
