## Build

### Linux
* Run ./deps.sh in command line.
* Run the following command.
* Install Qt.
```
./build-linux.sh
```

### Windows
* Install Qt(MinGW).
* Run the following command.
```
build-win.bat
```

### Android
* Install Qt supporting android.
* Run the following commands.
```
source ../android-build/android.profile
./build-android.sh
```
