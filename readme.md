## Build

### Linux
* Run ./deps.sh in command line.
* Run the following command.
* Install Qt.
```
make
```

### Windows
* Install Qt(MinGW).
* Run the following command.
```
build-win.bat
```

### Android
* Run the following commands.
```
source ../android-build/android.profile
./android-build.sh
```
